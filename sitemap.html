Globale Elemente. Top-Bar, Hauptnavigation, Kennenlernenformular und Footer erscheinen auf allen Seiten konsistent. Der "Kennenlernen"-Button öffnet immer das gleiche Formular.

→ Kennenlernenformular markiert jede Stelle, an der ein "Kennenlernen"-Button steht und das Formular geöffnet wird. Auf den Angebot-Karten wird das Formular mit dem passenden Anfragetyp vorausgewählt.

→ /pfad kennzeichnet Teaser-Sections auf der Landing, die direkt auf eine Unterseite verlinken. Drei Teaser → drei Unterseiten.

Anker. Anker (#einzel, #gruppe, #unternehmen, #faq, #contact) erlauben Deep-Links innerhalb einer Seite — z.B. vom Angebot-Teaser auf der Landing direkt zum passenden Block auf der Angebot-Subpage.
